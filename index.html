Daily Itinerary

每日行程

已按 9/28 14:30 梵蒂冈博物馆、9/30 上午博尔盖塞美术馆、10/2–10/5 voco Venice Mestre 住宿重排。

9/27 周日

香港 → 罗马｜抵达夜

傍晚抵达 FCO 后直接打车或预约接送进城。入住后简单晚餐；有精神再走 Campo de’ Fiori → 纳沃纳广场 → 万神殿外观。抵达夜不安排不可取消门票。

9/28 周一

圣彼得大教堂 + 梵蒂冈博物馆已订

07:30–08:00 抵达圣彼得广场，先看圣彼得大教堂；若排队和体力允许再登穹顶。12:00 在 Prati / Borgo 午餐。14:00 抵达博物馆入口，14:30 入场梵蒂冈博物馆，傍晚以西斯廷礼拜堂收尾。晚餐留在梵蒂冈附近。

9/29 周二

古罗马核心 + 万神殿约会晚餐

09:00 斗兽场，接古罗马广场和帕拉蒂尼山。13:00 Monti 区午餐，下午回酒店休息。17:00 卡比托利欧山、威尼斯广场，傍晚走到万神殿和纳沃纳广场。20:00 Armando al Pantheon 作为正式罗马晚餐；饭后夜游特莱维喷泉。

9/30 周三

博尔盖塞美术馆 → 佛罗伦萨

早上退房并把行李寄存在酒店，08:15 左右打车去博尔盖塞美术馆。首选 09:00–11:00 场次，重点看贝尔尼尼、卡拉瓦乔、拉斐尔、提香和卡诺瓦。参观后回酒店取行李，乘 13:20–14:20 间直达高铁前往佛罗伦萨。傍晚看大教堂外观、领主广场、老桥；晚餐 La Giostra。

10/1 周四

乌菲兹 + Oltrarno 慢逛 + 牛排

09:00 乌菲兹美术馆，集中看文艺复兴核心作品。午餐后休息，下午去 Santo Spirito、Oltrarno、老桥和阿诺河散步。学院美术馆改为可选。晚上 Buca Lapi，两人共享佛罗伦萨牛排。

10/2 周五

佛罗伦萨 → Venezia Mestre｜入住 voco

建议乘 10:00–10:40 间直达高铁，终点买到 Venezia Mestre，不要坐过头到 Santa Lucia。抵达后打车或公交去 voco Venice Mestre - The Quid by IHG，先办理入住和放行李。下午视体力从 Mestre 进岛，到 Venezia Santa Lucia / Piazzale Roma 后走 Cannaregio → Rialto → 圣马可夜景。第一晚以轻松 Cicchetti 为主，不要太晚返程。

10/3 周六

从 Mestre 早进岛｜圣马可 + 总督宫 + 贡多拉

07:30–08:00 从 voco 出发进岛，目标 08:30–08:45 抵达圣马可周边。09:00–10:00 圣马可大教堂，10:15–12:30 总督宫。因为酒店在 Mestre，中午不建议来回折返，改在岛上安排长午餐或咖啡休息。下午走 Dorsoduro、学院桥、Salute 教堂，日落前贡多拉。晚餐 Antiche Carampane 或 Al Covo 建议订 19:00–19:30，饭后返 Mestre。

10/4 周日

Mestre → Burano 彩色岛｜告别午餐

从 voco 早出发进岛，再从 Fondamente Nove 转水上巴士去 Burano，避免一天硬塞三岛。12:30 Al Gatto Nero da Ruggero 午餐，下午回主岛或直接回 Mestre。傍晚若还有体力，可在主岛做最后一次运河散步；若累了就在 Mestre 简单晚餐、整理行李，早点睡。

10/5 周一

Mestre → VCE → 香港

住在 Mestre 后不需要水上出租车。建议 05:30–05:45 从 voco 退房，提前请酒店预约陆路出租车或接送车去 Marco Polo Airport，目标 06:20–06:40 抵达机场。早班土航经伊斯坦布尔，10/6 清晨抵达香港。

10/6 周二

香港 → 深圳

取行李和入境后，再衔接跨境商务车、巴士或高铁回深圳。若约 07:40 抵港，上午至中午回到深圳较稳。

Transport

交通与通勤

境内只保留两段直达高铁。威尼斯住宿已改为 Mestre，大原则是：10/2 高铁买到 Venezia Mestre、每天早进岛晚返、10/5 清晨陆路去机场。

FCO → 罗马市区

抵达日建议官方出租车或预约接送。若精神和行李都轻，可搭 Leonardo Express 到 Termini 再打车。

两段高铁

9/30：Roma Termini → Firenze SMN，因新增博尔盖塞，建议 13:20–14:20 出发。
10/2：Firenze SMN → Venezia Mestre，10:00–10:40 出发，到站后打车或公交去 voco。

Mestre ↔ 威尼斯本岛

voco 不在本岛，也不在 Mestre 站正门旁。每天预留 20–30 分钟接驳到 Mestre 站或公交点，再进岛到 Venezia Santa Lucia / Piazzale Roma。晚上返程别排太晚。

10/5 Mestre → VCE

不再订水上出租车。建议 05:30–05:45 从 voco 出发，提前请酒店预约陆路出租车/接送车，目标 06:20–06:40 抵达 Marco Polo Airport。

威尼斯岛内移动

主岛内尽量步行；跨远距离用 ACTV 水上巴士。10/4 去 Burano 时从 Fondamente Nove 转船更顺。

行李策略

住 Mestre 的优势是不用拖箱过桥、坐水巴找酒店。10/2 到站后先回 voco 放行李，再轻装进岛。
